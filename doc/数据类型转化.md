在C++中，数字与字符串的相互转换以及二维数组的多重排序是常见操作。以下是经典实现方法：

### 1. 数字转字符串
```cpp
#include <string>
#include <sstream>

// 方法1：使用 std::to_string (C++11)
int num = 123;
std::string str1 = std::to_string(num); // "123"

// 方法2：使用 stringstream
double d = 3.14159;
std::stringstream ss;
ss << d;
std::string str2 = ss.str(); // "3.14159"
```

### 2. 字符串转数字
```cpp
#include <cstdlib>
#include <string>

// 方法1：使用 std::stoi/stod 系列 (C++11)
std::string str = "456";
int num1 = std::stoi(str);     // 456
double num2 = std::stod("3.14"); // 3.14

// 方法2：使用 stringstream
std::string input = "789";
int value;
std::stringstream(input) >> value; // 789

// 方法3：C风格 atoi/atof
const char* cstr = "123";
int num3 = std::atoi(cstr);      // 123
float num4 = std::atof("12.34"); // 12.34
```

### 3. 二维数组多重排序
```cpp
#include <vector>
#include <algorithm>

// 示例：对二维数组排序（每行3列）
// 规则：1. 按第0列升序 2. 第0列相同时按第1列降序 3. 前两列相同按第2列升序
std::vector<std::vector<int>> data = {
    {3, 2, 5},
    {1, 4, 6},
    {1, 4, 3},
    {2, 5, 1}
};

// 自定义排序函数
std::sort(data.begin(), data.end(), [](const std::vector<int>& a, const std::vector<int>& b) {
    if (a[0] != b[0]) 
        return a[0] < b[0];  // 第0列升序
    else if (a[1] != b[1])
        return a[1] > b[1];  // 第1列降序
    else 
        return a[2] < b[2];  // 第2列升序
});

/* 排序结果：
{1, 4, 6}
{1, 4, 3}
{2, 5, 1}
{3, 2, 5}
*/
```

### 4. 传统二维数组排序
```cpp
#include <algorithm>

int arr[][3] = {{3,2,5}, {1,4,6}, {1,4,3}, {2,5,1}};
int rows = sizeof(arr)/sizeof(arr[0]);

std::sort(arr, arr + rows, [](const auto& a, const auto& b) {
    if (a[0] != b[0]) return a[0] < b[0];
    if (a[1] != b[1]) return a[1] > b[1];
    return a[2] < b[2];
});
```

### 关键点说明：
1. **数字转字符串**
    - `std::to_string` 是最简洁方式（C++11起）
    - `stringstream` 支持格式控制（如精度设置）

2. **字符串转数字**
    - `std::stoxx` 系列提供异常安全（无效输入抛出异常）
    - C风格函数在转换失败时返回0（不报错）

3. **多重排序技巧**
    - 使用Lambda表达式定义自定义排序规则
    - 按优先级逐级比较字段
    - 注意比较方向（`<` 升序，`>` 降序）

4. **性能提示**
    - 对vector<vector>排序时，使用引用避免拷贝：`[](const auto &a, const auto &b)`
    - 传统数组排序需计算正确长度：`sizeof(arr)/sizeof(arr[0])`

### 完整示例（包含所有操作）：
```cpp
#include <iostream>
#include <vector>
#include <algorithm>
#include <string>
#include <sstream>

int main() {
    // 数字转字符串
    int num = 2023;
    std::string numStr = std::to_string(num);
    std::cout << "数字转字符串: " << numStr << "\n";

    // 字符串转数字
    std::string input = "3.14159";
    double pi = std::stod(input);
    std::cout << "字符串转数字: " << pi << "\n";

    // 二维数组多重排序
    std::vector<std::vector<int>> matrix = {
        {3, 1, 4},
        {1, 5, 9},
        {2, 6, 5},
        {1, 5, 2}
    };

    std::sort(matrix.begin(), matrix.end(), [](const auto& a, const auto& b) {
        if (a[0] != b[0]) return a[0] < b[0];    // 首列升序
        if (a[1] != b[1]) return a[1] > b[1];    // 次列降序
        return a[2] < b[2];                      // 末列升序
    });

    std::cout << "\n排序后矩阵:\n";
    for (const auto& row : matrix) {
        for (int val : row) 
            std::cout << val << " ";
        std::cout << "\n";
    }

    return 0;
}
```

输出：
```
数字转字符串: 2023
字符串转数字: 3.14159

排序后矩阵:
1 5 9 
1 5 2 
2 6 5 
3 1 4 
```
